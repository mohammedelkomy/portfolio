üöÄ Project Overview

This isn't just a data analysis project; it's a full-stack data science solution. I processed over 140,000 streaming records from my personal Spotify history to solve a behavioral question: "Can AI predict if I will skip a song before it finishes?"

‚öôÔ∏è The Technical Pipeline

  • 1. Advanced Data Cleaning (Python/Pandas):
    • Converted raw timestamps and handled missing values in playback context.
    • Logic Engineering: defined a "Skip" not just by the button press, but by calculating listening duration (e.g., < 30 seconds = Skip).
  • 2. Machine Learning (Scikit-Learn):
    • Built a Random Forest Classifier achieving 80% Accuracy.
    • Feature Engineering: Created features like Session_ID, Time_of_Day, and Track_Origin.
  • 3. Visualization (Power BI):
    • Designed a dark-themed UX mimicking the Spotify App.
    • Implemented Drill-down charts for Year/Month analysis and Heatmaps for activity tracking.

üí° Key Findings & Business Value

The model revealed a critical insight: Context > Content.

  • The "Shuffle" Effect: The #1 predictor for skipping was how the song started (Shuffle/Autoplay), not the Artist name.
  • Platform Behavior: 96% of streaming occurred on Android with a consistent 36.8% skip rate.
